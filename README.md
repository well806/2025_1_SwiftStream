# 2025_1_SwiftStream
Проект команды SwiftStream по курсу "Мобильный разработчик на iOS" от VK

## Описание проекта
**Название проекта** - BaumanCore 

**Цель проекта**: Создание мобильного приложения для студентов университета МГТУ. 

**Описание приложения**: BaumanCore — это удобное мобильное приложение-помощник для студентов МГТУ им. Баумана. Оно сопровождает студента на протяжении всего учебного периода и делает студенческую жизнь проще, предоставляя всю необходимую информацию, полезные сервисы и инструменты для комфорта и продуктивности.

Оно включает в себя:
* Личный профиль
* Расписание
* Навигатор
* Оценки, аналитика успеваемости
* Новости
